# Projeto usando Expo

Este é um projeto utilizando Expo para desenvolvimento de aplicações móveis multiplataforma.

## Como Iniciar

Siga os passos abaixo para começar a trabalhar no projeto:

### Pré-requisitos

Certifique-se de ter o Node.js e npm instalados. Você pode baixá-los em [nodejs.org](https://nodejs.org/).

### Instalação

1. Clone o repositório:

   ```bash
   git clone https://github.com/Ryan-Carloso/country-app-api
   ```

2. Instale as dependências do projeto:

   ```bash
   cd nome-do-projeto
   npm install
   ```

### Iniciando o Projeto

Para iniciar o projeto, você pode usar o Expo CLI:

```bash
npm start
```

ou

```bash
expo start
```

Isso iniciará o servidor de desenvolvimento do Expo. Você pode então escanear o código QR gerado usando o aplicativo Expo Go no seu dispositivo móvel para visualizar a aplicação em tempo real.

### Outros Comandos Úteis

- Para rodar a aplicação no emulador iOS:

  ```bash
  npm run ios
  ```

- Para rodar a aplicação no emulador Android:

  ```bash
  npm run android
  ```
